== Description ==

Easily find your medias on the solution of digital asset management ePhoto and import directly into WordPress.

Related Links:

* <a href="http://www.ephoto.fr/" title="Official website of the solution of digital asset management ePhoto">Official website of ePhoto"</a>
* <a href="http://blog.ephoto.fr/" title="Blog of the solution of digital asset management ePhoto">Official blog of ePhoto</a>

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Open the plugin configuration page, customize the buttons import
6. The ePhoto button is present in the richtext editor when you edit your pages and posts :)
